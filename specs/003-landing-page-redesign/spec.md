# Feature Specification: 랜딩페이지 디자인 수정 (커스텀 폰트 + 인터랙티브 글로브)

**Feature Branch**: `003-landing-page-redesign`
**Created**: 2026-03-03
**Status**: Draft
**Input**: User description: "랜딩페이지 디자인 수정 - eluofacevf.ttf 폰트 적용 및 인터랙티브 글로브 컴포넌트 통합"

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 브랜드 폰트로 타이틀 표시 (Priority: P1)

방문자가 랜딩 페이지에 접속했을 때, "ELUO AI SKILL HUB" 헤더 제목이 ELUO 전용 커스텀 폰트(eluofacevf.ttf)로 렌더링되어 브랜드 아이덴티티가 강조된다.

**Why this priority**: 브랜드 폰트는 첫인상을 결정짓는 핵심 시각 요소다. 페이지 로드 시 가장 먼저 눈에 들어오는 h1 타이틀에 커스텀 폰트를 적용하는 것은 모든 다른 디자인 변경의 기반이 된다.

**Independent Test**: 랜딩 페이지를 열어 h1 텍스트가 기본 시스템 폰트가 아닌 ELUO 브랜드 폰트로 렌더링되는지 시각적으로 확인하거나, 브라우저 개발자 도구에서 computed font-family를 확인함으로써 독립적으로 테스트 가능하다.

**Acceptance Scenarios**:

1. **Given** 방문자가 랜딩 페이지에 접속할 때, **When** 페이지가 완전히 로드되면, **Then** "ELUO AI SKILL HUB" h1 제목이 eluofacevf 커스텀 폰트로 렌더링되어야 한다.
2. **Given** eluofacevf.ttf 폰트 파일이 프로젝트에 존재할 때, **When** 브라우저가 페이지를 렌더링할 때, **Then** FOUT(Flash of Unstyled Text) 없이 또는 최소화하며 폰트가 적용되어야 한다.
3. **Given** 페이지 로드 시, **When** 폰트 파일 로드에 실패했을 때, **Then** 동일 계열의 대체 폰트(fallback)로 텍스트가 정상 표시되어야 한다.

### User Story 2 - 인터랙티브 글로브로 시각적 임팩트 제공 (Priority: P2)

방문자가 랜딩 페이지의 히어로 섹션에서 회전하는 3D 지구본을 보고 드래그로 조작할 수 있다. 전 세계 도시 간 연결 아크(arc)와 이동하는 점이 글로벌 플랫폼의 인상을 준다.

**Why this priority**: 인터랙티브 글로브는 기존의 텍스트 중심 히어로 섹션을 시각적으로 강화하여 방문자의 체류 시간과 관심도를 높인다. 폰트 적용 다음으로 시각적 임팩트가 크다.

**Independent Test**: 랜딩 페이지에서 지구본이 자동으로 회전하고, 마우스 드래그로 방향을 바꿀 수 있으며, 도시 마커와 연결 아크가 표시되는지 확인함으로써 독립적으로 테스트 가능하다.

**Acceptance Scenarios**:

1. **Given** 방문자가 랜딩 페이지에 접속할 때, **When** 히어로 섹션이 로드되면, **Then** 캔버스 위에 3D 지구본이 자동 회전하며 표시되어야 한다.
2. **Given** 지구본이 표시된 상태에서, **When** 방문자가 마우스로 드래그하면, **Then** 드래그 방향에 따라 지구본이 회전해야 한다.
3. **Given** 지구본이 표시된 상태에서, **When** 페이지가 로드되면, **Then** 도시 마커(San Francisco, London, Tokyo 등)와 도시 간 연결 아크가 표시되어야 한다.
4. **Given** 지구본이 표시된 상태에서, **When** 연결 아크가 렌더링되면, **Then** 아크 위를 이동하는 점(traveling dot)이 애니메이션되어야 한다.
5. **Given** 모바일 디바이스에서 접속할 때, **When** 히어로 섹션이 로드되면, **Then** 지구본이 적절한 크기로 조정되어 콘텐츠와 함께 세로 배치되어야 한다.

### User Story 3 - 새 레이아웃에서 기존 콘텐츠 접근 (Priority: P3)

방문자가 새롭게 디자인된 레이아웃에서도 기존의 서비스 소개 텍스트, 주요 기능 섹션, 시작하기 버튼에 원활하게 접근할 수 있다.

**Why this priority**: 레이아웃 변경 시 기존 콘텐츠(소개 텍스트, CTA 버튼, 기능 섹션)의 가독성과 접근성이 유지되어야 한다. 인터랙티브 요소가 추가되더라도 핵심 전환 경로가 방해받지 않아야 한다.

**Independent Test**: 새 레이아웃에서 "시작하기" 버튼 클릭, 서비스 소개 텍스트 가독성, 주요 기능 카드 표시 여부를 확인함으로써 독립적으로 테스트 가능하다.

**Acceptance Scenarios**:

1. **Given** 방문자가 새 레이아웃의 랜딩 페이지에 접속할 때, **When** 히어로 섹션이 표시되면, **Then** 서비스 소개 문구와 "시작하기" CTA 버튼이 명확히 표시되어야 한다.
2. **Given** 방문자가 페이지를 스크롤할 때, **When** 주요 기능 섹션에 도달하면, **Then** 스킬 검색, 원클릭 실행, 스킬 관리 카드가 정상 표시되어야 한다.
3. **Given** 방문자가 "시작하기" 버튼을 클릭할 때, **When** 버튼이 활성화되면, **Then** /login 페이지로 이동해야 한다.

---

### Edge Cases

- 저사양 기기(CPU/GPU 성능 제한)에서 캔버스 애니메이션이 프레임 드롭 없이 부드럽게 작동하는가?
- 모바일 터치 이벤트로 지구본 드래그가 정상 동작하는가?
- eluofacevf.ttf 파일을 브라우저가 로드하지 못할 경우 fallback 폰트로 레이아웃이 깨지지 않는가?
- 다크모드/라이트모드 환경에서 글로브 색상과 배경이 자연스럽게 보이는가?
- 히어로 섹션의 좌측 콘텐츠와 우측 글로브가 중간 크기 뷰포트(태블릿)에서 적절히 배치되는가?

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 랜딩 페이지 h1 제목("ELUO AI SKILL HUB")은 반드시 eluofacevf 커스텀 폰트로 렌더링되어야 한다.
- **FR-002**: eluofacevf 폰트는 현재 프로젝트에 포함된 `ELUOFACEVF.ttf` 파일을 사용하여 로드되어야 한다.
- **FR-003**: 폰트 로드 실패 시 동일 계열 sans-serif 대체 폰트로 fallback이 적용되어야 한다.
- **FR-004**: 랜딩 페이지 히어로 섹션에 인터랙티브 3D 지구본 컴포넌트가 표시되어야 한다.
- **FR-005**: 지구본은 페이지 로드 직후 자동으로 회전해야 한다.
- **FR-006**: 방문자가 지구본을 마우스 드래그(또는 터치)로 조작하여 회전 방향을 바꿀 수 있어야 한다.
- **FR-007**: 지구본에 전 세계 주요 도시 마커(최소 10개)와 도시 간 연결 아크가 표시되어야 한다.
- **FR-008**: 연결 아크 위에는 이동하는 점 애니메이션이 실행되어야 한다.
- **FR-009**: 히어로 섹션은 좌측 텍스트 콘텐츠(서비스 소개, CTA 버튼)와 우측 지구본이 2단 레이아웃으로 배치되어야 한다.
- **FR-010**: 모바일(768px 미만) 환경에서는 텍스트 콘텐츠가 위, 지구본이 아래에 세로 배치되어야 한다.
- **FR-011**: 기존의 주요 기능 섹션과 푸터는 새 레이아웃에서도 그대로 유지되어야 한다.
- **FR-012**: 지구본 컴포넌트는 `/components/ui/interactive-globe.tsx` 경로에 위치해야 한다(또는 프로젝트의 공용 UI 컴포넌트 경로).

### Assumptions

- 프로젝트에 `ELUOFACEVF.ttf` 파일이 이미 `src/app/font/` 경로에 존재하는 것으로 확인됨.
- 지구본 컴포넌트는 외부 3D 라이브러리 없이 Canvas API만으로 구현됨 (별도 패키지 설치 불필요).
- 히어로 섹션 배경색은 기존 `bg-brand-navy`를 유지하거나 데모 디자인의 `bg-card`를 참고하여 조정함.
- 글로브 기본 색상(dotColor, arcColor, markerColor)은 브랜드 컬러에 맞게 조정 가능.

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 랜딩 페이지 로드 후 1초 이내에 h1 제목이 eluofacevf 폰트로 렌더링된다.
- **SC-002**: 지구본 애니메이션이 최소 30fps로 부드럽게 실행된다.
- **SC-003**: 데스크탑(1024px 이상)과 모바일(375px)에서 레이아웃 깨짐 없이 정상 표시된다.
- **SC-004**: "시작하기" CTA 버튼이 새 레이아웃에서도 히어로 섹션 내 명확하게 표시되며 클릭 가능하다.
- **SC-005**: 지구본 드래그 인터랙션이 마우스 및 터치 입력 모두에서 즉각 반응한다(지연 100ms 이하).
