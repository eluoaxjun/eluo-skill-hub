# Product Overview

Eluo Skill Hub는 웹 에이전시의 기획-디자인-퍼블리싱-개발-QA 워크플로우를 자동화하는 **플러그인 마켓플레이스**이다.
Claude Code 플러그인(Skill) 시스템 기반으로, 각 직군별 반복 업무를 자동화하는 스킬을 등록·검색·설치·실행할 수 있다.

## Core Capabilities

1. **스킬 마켓플레이스** — 스킬 등록, 검색, 설치, 버전 관리
2. **직군별 워크플로우 자동화** — 기획/디자인/퍼블리싱/개발/QA 각 단계에 특화된 스킬 제공
3. **스킬 실행 환경** — Claude Code CLI 기반 스킬 실행 및 결과 관리
4. **사용자 협업** — 스킬 공유, 리뷰, 평점 시스템을 통한 커뮤니티 기반 품질 관리

## Target Use Cases

- **기획자**: 요구사항 정리, 와이어프레임 생성, PRD 작성 자동화
- **디자이너**: 디자인 토큰 추출, 컴포넌트 명세 생성, 접근성 검증
- **퍼블리셔**: HTML/CSS 변환, 반응형 레이아웃 생성, 크로스브라우저 검증
- **개발자**: API 스캐폴딩, 테스트 코드 생성, 코드 리뷰 자동화
- **QA**: 테스트 시나리오 생성, 회귀 테스트 자동화, 버그 리포트 생성

## Value Proposition

- **단일 플랫폼**: 에이전시 전체 직군이 하나의 플랫폼에서 자동화 스킬을 공유
- **Claude Code 네이티브**: CLI 환경에서 즉시 사용 가능한 스킬 시스템
- **커뮤니티 기반 확장**: 누구나 스킬을 등록하고 공유할 수 있는 오픈 생태계

## Domain Model (Bounded Contexts)

| 바운디드 컨텍스트 | 책임 |
|---|---|
| **Skill Catalog** | 스킬 메타데이터, 검색, 카테고리 관리 |
| **Skill Registry** | 스킬 등록, 버전 관리, 퍼블리싱 |
| **User Account** | 사용자 인증, 프로필, 권한 관리 |
| **Review & Rating** | 스킬 리뷰, 평점, 신고 |
| **Installation** | 스킬 설치, 업데이트, 의존성 관리 |

## User Personas

| 페르소나 | 역할 | 주요 니즈 |
|---|---|---|
| **스킬 소비자** | 스킬을 검색하고 설치하여 사용 | 빠른 검색, 신뢰할 수 있는 스킬, 쉬운 설치 |
| **스킬 제작자** | 스킬을 개발하고 등록 | 간편한 등록 프로세스, 버전 관리, 사용 통계 |
| **플랫폼 관리자** | 마켓플레이스 운영 및 품질 관리 | 스킬 심사, 사용자 관리, 통계 대시보드 |
